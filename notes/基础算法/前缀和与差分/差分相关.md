# 有关于差分思想
## 先来给出二维差分的经典例子吧：  
```c++
#include <bits/stdc++.h>
using namespace std;
const int N=1e3+10;
typedef long long ll;
ll a[N][N],d[N][N];
int main()
{
    ll n,m,q;
    cin>>n>>m>>q;
    for(int i=1;i<=n;i++)
    {
        for(int j=1;j<=m;j++)
        {
            cin>>a[i][j];
            d[i][j]=a[i][j]-a[i-1][j]-a[i][j-1]+a[i-1][j-1];
        }
    }
    while(q--)
    {
        ll x1,y1,x2,y2,c;
        cin>>x1>>y1>>x2>>y2>>c;
        d[x1][y1]+=c;
        d[x2+1][y1]-=c;
        d[x1][y2+1]-=c;
        d[x2+1][y2+1]+=c;
    }
    //最后求一遍前缀和复原变化后的数组
    for(int i=1;i<=n;i++)
    {
        for(int j=1;j<=m;j++)
        {
            a[i][j]=d[i][j]+a[i-1][j]+a[i][j-1]-a[i-1][j-1];
            cout<<a[i][j]<<" ";
        }
        cout<<"\n";
        
    }
    return 0;
}
```
## 差分思想
通过上面那个二维差分的例子，我们很容易得出差分的核心思想
差分主要用于解决改变某一区间段的值的问题，与前缀和略有不同，但也存在联系   
我们是通过给出差分数组：a[i]-a[i-1]，然后在差分数组上做出改变，最后求前缀和来实现目的的

## 二维差分的易错点
我认为二维差分在求差分数组时容易出错   
**关键公式**：
**q[i][j]=a[i][j]-a[i-1][j]-a[i][j-1]+a[i-1][j-1]**
### 这个是怎么来的呢
我们可以通过逆向思维来考虑：  
- 我们的目的是通过差分数组进行前缀和来实现目的的
- 前缀和公式为a[i][j]=a[i-1][j]+a[i][j-1]-a[i-1][j-1]+d[i][j];
- 所以我们进行简单的移项就能得出差分数组怎么算
