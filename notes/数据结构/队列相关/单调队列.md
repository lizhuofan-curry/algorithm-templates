# 用数组模拟单调队列解决滑动窗口问题  
``` c++
#include <bits/stdc++.h>
using namespace std;
const int N=1e6+10;
int a[N];
int main()
{
    cin.tie(0);
    int n,k;
    cin>>n>>k;
    for(int i=0;i<n;i++) cin>>a[i];
    int q[N],tt=-1,hh=0;
    for(int i=0;i<n;i++)
    {
        //去除过期元素，每次只移动一格，所以用if就够了
        if(tt>=hh && i-k+1>q[hh]) hh++;
        //清理没用的元素，这里要找的是最小的，所以把在左边的且小于等于x从队列中删除
        while(tt >=hh &&a[q[tt]] >=a[i]) tt--;
        q[++tt]=i;
        if(i>=k-1) cout<<a[q[hh]]<<" ";
    }
    cout<<"\n";
    tt=-1,hh=0;
    for(int i=0;i<n;i++)
    {
        //去除过期元素，每次只移动一格，所以用if就够了
        if(tt>=hh && i-k+1>q[hh]) hh++;
        //清理没用的元素，这里要找的是最大的，所以把在左边的且小于等于x从队列中删除
        while(tt >=hh && a[q[tt]]<=a[i]) tt--;
        q[++tt]=i;
        if(i>=k-1) cout<<a[q[hh]]<<" ";
    }
    return 0;
}
```
***  
## 注意易错点 ：  
- 这里首先要判断区间是否过期，若过期则需要更新头指针
- 然后通过判断条件来清理里面无用的数据
- 输出时的判断条件记得是i>=k-1,意思是第一次移动到区间长度为k后每次移动都要输出一次了
- 还有一个细节点就是你要先**更新**队列再输出，万一你把队列里面的数都**删除**了，然后输出，能输出什么
