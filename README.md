# Algorithm Templates

这个仓库用于记录我在学习算法过程中遇到的常见模板、经典题型和易错点。

目前我主要使用 C++ 学习算法，并跟随 AcWing 算法基础课进行系统训练。虽然现在积累的内容还不多，但我希望通过这个仓库，把每天学到的知识一点点沉淀下来。

## 为什么创建这个仓库？

学习算法的过程中，很多模板会反复出现，例如：

- 基础算法
  - [排序合集](basic/排序)
  - [二分查找](basic/二分查找板子.cpp)
  - [高精度合集](basic/高精度)
  - [前缀和与差分](basic/前缀和与差分)
  - [双指针](basic/双指针)
  - [位运算](basic/位运算)
  - [离散化](basic/离散化)
  - [区间合并](basic/区间合并)
- 数据结构
  - [单链表](数据结构/链表实现/单链表)
- 并查集
- BFS / DFS
- 动态规划
- 图论算法
- 数学相关模板

如果只是刷完题就结束，很容易过几天就忘记。  
所以我希望把这些内容整理成自己的模板库，方便以后复习、查阅和总结。

这个仓库不仅是代码记录，也是一份学习轨迹。

## 我的目标

- 坚持整理常见算法模板
- 记录每个模板的适用场景
- 总结刷题时遇到的易错点
- 提高代码熟练度和算法理解能力
- 为后续竞赛、考试和面试打基础
