# 双指针算法中遇到的问题  
板子如下：  
```c
for (int i = 0, j = 0; i < n; i ++ )
{
    while (j < i && check(i, j)) j ++ ;

    // 具体问题的逻辑
}
常见问题分类：
    (1) 对于一个序列，用两个指针维护一段区间
    (2) 对于两个序列，维护某种次序，比如归并排序中合并两个有序序列的操作
```
### 它有三个经典应用题型
- 最长不连续子序列
- 数组元素的目标和
- 判断子序列
## 1.最长不连续子序列易错点  [题目链接](https://www.acwing.com/problem/content/801/)
它的核心思想是固定左端点不动，让右端点动  
将a[i]中的数用数组s[i]来记录  
在不符合条件时，让左端点动起来，使其符合条件  
用max(res,i-j+1)来刷新答案    
## 2.数组元素的目标和  [题目链接](https://www.acwing.com/problem/content/description/802/)
它的核心思想是让其中一个数组从左端开始，另外一个数组从右端开始移动  
其中有一点点二分相关的思想  
能这么做的原因有这两个数组都是有序排列的
## 3.判断子序列  [题目链接](https://www.acwing.com/problem/content/2818/)
它的核心思想是当且仅当符合条件时a数组才向右移动，否则只有b数组单独移动    
循环的判断条件有i<n && j<m  
所以当j右移一定次数时循环结束  
输出的判断条件是i的大小
