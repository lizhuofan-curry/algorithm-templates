# 数据结构之并查集
```c++
#include <bits/stdc++.h>
using namespace std;
const int N=1e5+10;
int q[N];
int find(int x)
{
  //压缩路径
  if(q[x]!=x) q[x]=find(q[x]);
  return q[x];
}
int main()
{
  int n,m;
  cin>>n>>m;
  for(int i=1;i<=n;i++) q[i]=i;
  while(m--)
    {
      char op;
      int a,b;
      cin>>op>>a>>b;
      if(op=='M') q[find(a)]=find(b);
      if(op=='Q') 
      {
        if(find(a)==find(b))
          cout<<"Yes"<<"\n";
        else cout<<"No"<<"\n";
      }
    }
  return 0;
}
```
## 注意点：
- 并查集代码比较短，也比较好理解
- 在find函数中找祖宗节点记得用if,用while容易进入死循环
- 巧妙点是在寻找祖宗节点的同时压缩路径了（让该集合中的所有结点都指向了祖宗节点）
